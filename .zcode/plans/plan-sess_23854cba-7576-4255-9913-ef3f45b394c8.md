## Rencana: Hapus Semua Gradient, Blob, dan Emoji

> Catatan: Tidak ditemukan blob component atau emoji di codebase. Fokus utama adalah menghapus semua gradient.

### File yang akan diubah:

#### 1. `frontend/src/app/globals.css`
- **Hapus class `.glass-card`** — ganti `linear-gradient(135deg, ...)` background menjadi solid `background-color` menggunakan `var(--card)`
- **Hapus class `.icon-gradient`** — ganti `linear-gradient(135deg, ...)` background menjadi solid warna (gunakan `var(--accent-start)` atau warna slate solid)
- **Hapus class `.gradient-border-top`** — ganti gradient border menjadi solid warna border
- **Hapus class `.progress-bar-inline .fill`** — ganti gradient menjadi solid warna (misal `var(--accent-start)`)
- **Hapus CSS custom properties** `--accent-start`, `--accent-end`, `--accent-light`, `--accent-surface` jika sudah tidak digunakan setelah perubahan

#### 2. `frontend/src/app/login/page.tsx`
- Ganti `bg-gradient-to-br from-slate-900 via-slate-800 to-slate-900` menjadi `bg-slate-900` (solid color)

#### 3. `frontend/src/app/signup/page.tsx`
- Sama seperti login — ganti gradient background menjadi solid color

#### 4. `frontend/src/components/layout/sidebar.tsx`
- **Logo area accent line**: ganti `linear-gradient(90deg, var(--accent-start), var(--accent-end), transparent)` menjadi solid color border
- **Sidebar background**: ganti `linear-gradient(180deg, ...)` menjadi solid color `hsl(222.2 47.4% 11.2%)`
- **Active nav item**: ganti `linear-gradient(135deg, #334155, #475569)` menjadi solid background `bg-slate-700` atau `#475569`

#### 5. `frontend/src/components/layout/topbar.tsx`
- **Bottom accent line**: ganti `linear-gradient(90deg, transparent, ..., transparent)` menjadi solid border bawah
- **Avatar gradient**: ganti `linear-gradient(135deg, ...)` menjadi solid warna background

#### 6. `frontend/src/app/dashboard/page.tsx`
- **Chart SVG bars**: Hapus semua `<LinearGradient>` definitions dan `<defs>` sections
- Ganti `url(#gradientName)` fill references pada `<rect>` elements menjadi solid color fills
- Chart colors yang ada: RFS (#475569), NY (#94a3b8), Category (#475569), Issue (#78716c), NOP (#334155), Mitra (#475569)
- **Pie chart**: Tidak ada gradient, sudah solid — tidak perlu diubah

#### 7. `frontend/src/components/charts/dynamic-chart.tsx`
- Periksa apakah ada gradient di komponen chart dinamis — hapus jika ada

### Prinsip perubahan:
- Semua `linear-gradient` diganti dengan **solid color** yang sesuai (warna awal gradient atau warna netral yang cocok)
- Tidak ada penambahan warna baru — gunakan warna yang sudah ada di palette (slate/stone)
- Pastikan dark mode tetap berfungsi dengan benar
- Pastikan visual tidak menjadi kurang menarik — gunakan solid color yang tetap profesional
