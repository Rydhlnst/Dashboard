# UI Redesign Plan: Dashboard Overview — "Modern Teal Gradient"

## Tujuan
Mengubah Dashboard Overview dari tampilan shadcn/ui generic menjadi desain yang lebih modern, memorable, dan khas dengan tema **Modern Teal Gradient** — tetap profesional tapi dengan depth, glassmorphism, dan micro-interactions.

## File yang Akan Diubah

### 1. `frontend/src/app/globals.css`
- Tambah CSS custom properties untuk gradient stop colors (`--gradient-start`, `--gradient-end`, dll)
- Tambah utility classes baru:
  - `.glass-card` — backdrop-blur + semi-transparent background
  - `.gradient-border-top` — gradient accent di top border
  - `.kpi-glow` — subtle glow shadow untuk KPI cards
  - `.hover-lift` — hover translate-Y + shadow transition
- Update chart text styling
- Tambah custom scrollbar untuk dark mode yang lebih halus

### 2. `frontend/src/app/layout.tsx`
- Import font `JetBrains Mono` dari next/font/google untuk data/numbers
- Tambah ke className body

### 3. `frontend/src/components/ui/kpi-card.tsx`
- Tambah `accentColor` prop (warna per tipe KPI)
- Icon wrapper: gradient background (teal→cyan) dengan glow shadow
- Tambah `border-l-4` dengan accent color
- Hover effect: `-translate-y-0.5` + shadow yang membesar
- Value: gunakan `font-mono` class untuk angka (JetBrains Mono)
- Optional: mini progress ring untuk KPI "Progress %"

### 4. `frontend/src/app/dashboard/page.tsx`
- **ChartCard wrapper**: upgrade dari flat card ke glass card dengan gradient top border
- **Hero chart (RFS vs Remaining)**: bar fill menggunakan linear gradient, chart height lebih besar
- **KPI Row**: berikan accentColor yang sesuai per card (teal=completed, amber=remaining, rose=issues, dll)
- **Top Pending Sites table**: mini progress bar inline di kolom progress, highlight row pertama
- **Section headers**: tambahkan subtle visual separator
- **Spacing**: sedikit tambah breathing room antar section

### 5. `frontend/src/components/layout/topbar.tsx`
- Glass effect diperkuat: `backdrop-blur-xl` + opacity lebih rendah
- Tambah subtle gradient line di bawah

### 6. `frontend/src/components/layout/sidebar.tsx`
- Active nav item: gradient background + subtle glow, bukan flat teal
- Logo area: tambahkan accent gradient line
- Hover transitions yang lebih halus

## Design Tokens Baru

```
Color additions:
- --gradient-teal-start: #0f766e  (teal-700)
- --gradient-teal-end: #14b8a6    (teal-500)  
- --gradient-teal-light: #ccfbf1  (teal-100)
- --glow-teal: 0 8px 32px rgba(20, 184, 166, 0.15)

Font:
- JetBrains Mono (via next/font) untuk angka/KPI values

New utilities:
- .glass-card → backdrop-blur-md bg-white/70 border-white/20
- .hover-lift → hover:-translate-y-0.5 hover:shadow-lg transition-all duration-200
- .gradient-border-top → border-image linear-gradient(to right, teal-500, cyan-400) 1
```
